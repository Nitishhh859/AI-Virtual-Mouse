## 📌 Overview

A real-time **AI-powered virtual mouse system** that allows users to control their computer cursor using hand gestures.
This project leverages **MediaPipe hand tracking** and **OpenCV** to deliver a smooth and responsive interaction experience.

---

## ✨ Features

* 🎯 **Cursor Control** using index finger
* 🖱️ **Click Detection** via thumb-index pinch
* ⚡ **Real-time Performance** with optimized tracking
* 🎥 **Live Hand Tracking** using MediaPipe
* 🔄 **Smooth Movement** using interpolation

## ⚙️ Installation

### 1️⃣ Clone Repository

```bash
git clone https://github.com/your-username/gesture-controlled-mouse.git
cd gesture-controlled-mouse
```

### 2️⃣ Create Virtual Environment

```bash
python -m venv gesture_env
gesture_env\Scripts\activate
```

### 3️⃣ Install Dependencies

```bash
pip install -r requirements.txt
```
